Projects

Project Name: OOUHEALTHCARE

This project focused on the development of an electronic health record management system to improve Olabisi Onabanjo University health record management service. This represents a transition from the traditional paper-based patient record keeping to a more sophisticated electronic/computer-based format. The developed system improved the Olabisi Onabanjo University’s health care record management, which reduced problems associated with patients’ records keeping and also increased the rate at which data is accessed, thus improving healthcare service delivery and achieving greater turnaround time.

Project Name: EXPERT DIAGNOSTIC SYSTEM

This project provided a reliable expert diagnostic system to improve Bowen University Hospital level of health care services offered and reducing the work load of medical doctors attending to so many patients in limited time. It supports growth and improves the medical record system of a hospital. Provides a well vast knowledge of the diseases involved for better diagnosis. Provides a confidential and friendly user interface. Provides automated records of consultation visits made to the hospital. Provides a facility that helps non-expert personnel solve problems that require some expertise.
